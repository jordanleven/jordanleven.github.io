Nedryland

Nedryland is my personal collection of dotfiles that I use to provision new computers and synchronize my command line interface preferences. This allows me to make sure each of my computers are using the same set of Zsh (Z Shell), Starship, and Oh My Zsh customizations. Nedryland includes a series of time-saving Git workflow improvements, like automatically attaching ticket numbers to commits, commands for generating and zipping patches, and pruning stale branches.

Themes for iTerm2, Slack, and Xcode are carefully created using a color pallette inspired by Jurassic Park.

Nedryland is an open-source and is available for Oh My Zsh and Jurassic Park fans alike.
