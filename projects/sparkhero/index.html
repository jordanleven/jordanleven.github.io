SparkHero!

May 2022

What better way to learn how to build pure-JavaScript web applications without the use of a UI framework? While I co-led Sparkbox's Full Stack Developer program, I was tasked with improving and streamlining the apprenticeship curriculum. Enter SparkHero.

SparkHero is an exercise that challenges apprentices to build a JavaScript app with ready-to-go ReSTful APIs that show off fan-favorite Marvel and DC characters. It covers all of the same disciplines that were previously covered in ad hoc lessons on building web applications but in a way that provides more structure and more closely resembled how real client projects were led. Throughout this two-weeks exercise, apprentices were tasked with adding new features like adding support for fetching data from API endpoints, sorting, and filtering.

This project comes preloaded with three ReSTful API endpoints compatible with Netlify serverless functions, Express, WebPack, Jest, Sass, a complete set of linters, and a customized CI/CD pipeline to automate deployments to Netlify for both PRs and code merged into the production branch.

Still curious? See SparkHero! yourself!
