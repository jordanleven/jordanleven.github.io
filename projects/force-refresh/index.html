Force Refresh

Force Refresh is a lightweight WordPress plugin born out of a simple necessity: the ability to force a user to refresh their browser on single-page applications. This plugin allows site owners to request users that are currently connected to their website by storing current site and page versions as hexadecimal representations of the time the refresh was requested. Browsers then periodically check to see if the stored site version corresponded to the current site version and, if not, refresh the page.

Force Refresh is an open-source plugin with a custom Vue-powered admin front-end that uses a series of WordPress-flavored HTTP APIs for authentication and storing current versions of the site.
