Relational Data and MySQL Lesson

March 2022

The final lesson in the Sparkbox Full Stack Apprenticeship was going over the basics of MySQL and relational data. By combining my professional interest in teaching relational data with my personal interest in the film industry, I was able to create a fun and practical hour-long lesson and practice database. Keeping with the movie theme set by the famous Sakila sample database, the custom lesson and accompanying Universal Studios-themed database allowed apprentices to understand the relationship between data.

This lesson covered many MySQL basics, like data normalization, joins, as well as primary, foreign, composite, and surrogate keys. Additionally, this lesson taught apprentices how to read and write entity relationship diagrams (ERDs), queries, stored procedures, and stored functions.
